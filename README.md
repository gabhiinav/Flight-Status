# Flight Status

Tech Stack:

- Next.js
- React
- Typescript
- Express
- SocketIO
- Web Push
- PostgreSQL

### Setup .env

```js
PORT=
DATABASE_URL=
VAPID_PUBLIC_KEY=
VAPID_PRIVATE_KEY=
VAPID_SUBJECT=
```

### Start Backend

```shell
npm run dev
```

### Start Frontend

```shell
npm run dev
```
